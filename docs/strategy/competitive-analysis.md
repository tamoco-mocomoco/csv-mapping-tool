# 競合分析: CSV-Mapping Tool

## 概要

CSV-Mapping Toolは「ブラウザ完結・無料・OSS」のCSVカラムマッピング&変換ツールである。本ドキュメントでは、代替手段となる3つの競合カテゴリとの比較を通じて、差別化ポイントを明確にする。

---

## 競合カテゴリ比較

### 比較対象

| カテゴリ | 代表的なツール/サービス |
|---|---|
| オンラインCSV変換ツール | ConvertCSV, CSV Lint, Mr. Data Converter, RebaseData |
| ETL/データ変換ツール | Talend Open Studio, Pentaho Data Integration, Alteryx, ASTERIA Warp |
| Excel/スプレッドシート | Microsoft Excel, Google Sheets, LibreOffice Calc |

## カテゴリ別の詳細比較

### 1. vs オンラインCSV変換ツール

**競合の強み:**
- 特定のフォーマット変換（JSON, XML等への変換）に特化
- シンプルで迷いにくい

**競合の弱み:**
- カラム単位のマッピング機能がない
- データをサーバーにアップロードする必要があり、機密データを扱えない
- 変換設定を保存・再利用できない
- コンバーターの組み合わせができない

**CSV-Mapping Toolの優位性:**
- 「カラムマッピング + 変換」という組み合わせが唯一無二
- ローカル処理のためデータが外部に漏れない
- プロファイル保存で繰り返し作業を効率化

---

### 2. vs ETL/データ変換ツール

**競合の強み:**
- 大規模データ処理、DB接続、API連携など圧倒的な機能の幅
- スケジュール実行、バッチ処理が可能
- 企業レベルのデータパイプライン構築に対応

**競合の弱み:**
- インストールと環境構築が必要
- 学習コストが非常に高い（研修・教育が必要）
- 有料ツールは高額なライセンス費用
- 「CSVのカラム変換」だけのためにはオーバースペック

**CSV-Mapping Toolの優位性:**
- URLを開くだけで即使える（導入コストゼロ）
- 「CSVカラムのマッピングと変換」に特化しており、迷わず使える
- 非エンジニアでも直感的に操作可能
- ETLツールの導入を待たずに、現場レベルで即解決できる

---

### 3. vs Excel/スプレッドシート

**競合の強み:**
- ほぼすべてのPCにインストール済みで、習熟者が多い
- 自由度が高く、あらゆる加工が可能
- ピボットテーブル、グラフなど分析機能も豊富

**競合の弱み:**
- カラムマッピングは手作業（VLOOKUP、INDEX/MATCH等の関数知識が必要）
- 手順が属人化しやすく、引き継ぎが困難
- ミスが発生しやすい（コピペミス、関数の参照ズレ）
- SJIS/UTF-8の文字コード変換が煩雑
- 同じ変換を繰り返す場合、毎回手作業が発生

**CSV-Mapping Toolの優位性:**
- マッピングルールが視覚的に明示され、属人化しない
- プロファイル保存により、誰でも同じ変換を再現可能
- SJIS/UTF-8の切り替えがワンクリック
- 関数の知識不要で、コンバーターを選ぶだけ

---

## 差別化サマリー

CSV-Mapping Toolが占める独自のポジションは以下の通りである。

| 差別化軸 | 内容 |
|---|---|
| **ニッチ特化** | 「CSVカラムのマッピングと変換」という明確なユースケースに絞り込み |
| **導入障壁ゼロ** | インストール不要、無料、ブラウザだけで完結 |
| **データ安全性** | 全処理がブラウザ内で完結し、データがサーバーに送信されない |
| **再現性** | プロファイル保存により、変換ルールをチーム内で共有・再利用可能 |
| **柔軟な変換** | 11種類のコンバーターをパイプラインで自由に連鎖できる |
| **日本語環境対応** | SJIS/UTF-8の切り替え対応（日本のビジネス現場で頻出する要件） |

### ポジショニング

```
       高機能
        |
  ETL/データ変換ツール
        |
        |
        |
低コスト ----+---- CSV-Mapping Tool ---- 高コスト
        |
        |
  Excel     オンラインCSV変換ツール
        |
       低機能
```

CSV-Mapping Toolは「低コスト × 中機能」のポジションに位置し、ETLツールほどの投資は不要だが、Excel手作業やオンラインツールでは実現しにくい「カラムマッピング + 変換パイプライン + 設定再利用」を提供する。
